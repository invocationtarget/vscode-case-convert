# Case convert
This extension is intended to cover case convertion during developing
---

Currently supported convertions:
- cammelCase
- SNAKE_CASE
- kebab-case
- UPPERCASE
- lowercase
- PascalCase
- Sentence case
- Start Case
- CONSTANT_CASE
- lo_dash_space_case
- dash-space-case
- space-case
